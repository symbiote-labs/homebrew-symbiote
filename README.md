# homebrew-symbiote

Homebrew tap for the **Symbiote CLI** (`symbiote`).

```sh
brew tap symbiote-labs/symbiote
brew install symbiote
```

The `Formula/symbiote.rb` formula is published automatically by the release
pipeline; it points at the binaries in
[symbiote-cli-dist releases](https://github.com/symbiote-labs/symbiote-cli-dist/releases).
Works on macOS and Linux (Linuxbrew).
